Device information
------------------
.. screenshot:: http://localhost:5173/device
  :browser: chromium
  :viewport-width: 1280
  :viewport-height: 960
  :color-scheme: dark
  :status-code: 200,302

This page gives some health information about your device. It will show the amount of memory used and available,
disk space, the device's hostname and current IP-address.
